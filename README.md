# 🎧 Mood-Based Song Recommendation System

The Mood-Based Song Recommendation System is a web-based application designed to recommend songs based on the user’s emotional state. Instead of using face recognition, the system identifies the user’s mood through a simple questionnaire-based approach, making it more privacy-friendly and user-focused.

Based on the selected mood, the application generates a personalized playlist that matches the user’s emotions and preferences.

---

## 🚀 Features

- Questionnaire-based mood detection  
- Multiple mood categories (Happy, Sad, Calm, Energetic, Bored, etc.)  
- Personalized song recommendations  
- “Surprise Me” option for random playlist generation  
- Simple and user-friendly interface  
- No camera or facial data required

---

## 🎯 Objective

The main objective of this project is to create a lightweight and privacy-safe music recommendation system that enhances user experience by understanding emotions through user input rather than invasive technologies.
